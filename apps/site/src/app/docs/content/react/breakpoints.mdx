# Breakpoints - Material Tailwind

`@material-tailwind/react` is using the default breakpoints from Tailwind CSS. The breakpoints are used to create responsive designs for different screen sizes.

---

## Default Breakpoints

<CodePreview>
```js
{
  'sm': '640px',
  'md': '768px',
  'lg': '1024px',
  'xl': '1280px',
  '2xl': '1536px',
}
```
</CodePreview>

<br />

For more information about Tailwind CSS breakpoints customization, please check the <Link href="https://tailwindcss.com/docs/screens?ref=material-tailwind" target="_blank">Tailwind CSS Official Documentation</Link>
